## 🎯 Proje Amacı

Bu proje, Titanic veri seti kullanılarak yolcuların hayatta kalma durumunu tahmin etmeyi amaçlar.  
Makine öğrenmesi modeli olarak Logistic Regression kullanılmıştır ve Streamlit ile web arayüzü geliştirilmiştir.

---

## 🧠 Makine Öğrenmesi Süreci

- Veri temizleme ve ön işleme
- Eksik verilerin doldurulması
- Kategorik verilerin sayısallaştırılması
- Logistic Regression modeli ile eğitim
- Tahmin işleminin gerçekleştirilmesi

---

## 🛠️ Kullanılan Teknolojiler

- Python  
- Pandas  
- Scikit-learn  
- Streamlit
