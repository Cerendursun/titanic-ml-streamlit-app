# Titanic Hayatta Kalma Tahmini

Bu proje, Titanic veri seti kullanılarak yolcuların hayatta kalma durumunu tahmin eden bir makine öğrenmesi uygulamasıdır. Model olarak Logistic Regression kullanılmıştır ve arayüz Streamlit ile oluşturulmuştur.

## Kullanılan Teknolojiler
- Python
- Pandas
- Scikit-learn
- Streamlit

## Proje İçeriği
- Veri temizleme ve ön işleme
- Eksik verilerin doldurulması
- Kategorik verilerin sayısal hale getirilmesi
- Logistic Regression modeli ile tahmin
- Streamlit ile basit web arayüzü
